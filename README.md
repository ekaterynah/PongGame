# Pong Game + Boost

## Опис гри

**Pong + Boost** — це навчальна 2D-гра, створена на основі класичної аркадної гри **Pong**, реалізована мовою **Python** з використанням бібліотеки **pygame**.

На ігровому полі розташовані дві ракетки (ліва та права) і м’яч, який безперервно рухається між ними.
Гравець керує лівою ракеткою та відбиває м’яч, змагаючись проти AI суперника, або можна встановити ручний формат гри для другого гравця.  
Після кожного пропущеного м’яча суперник отримує одне очко, а м’яч повертається у центр поля та починає новий раунд.
Гра триває до моменту, поки один із гравців не набере встановлену кількість очок.

---

## Керування

| Клавіша | Дія |
|-------|-----|
| **W / S** | Рух лівої ракетки вгору / вниз |
| **SHIFT (утримувати)** | Ручне керування правою ракеткою поки утримується клавіша|
| **↑ / ↓** | Рух правої ракетки (у ручному режимі) |
| **ENTER** | Початок гри з меню |
| **P** | Пауза / продовження гри |
| **ESC** | Вихід з гри |

---

## Особливості та покращення

У порівнянні з класичною версією **Pong**, у грі реалізовано декілька покращень:

- **Power-up “Speed Boost”**  
  На полі періодично з’являється бонус. Якщо м’яч торкається його, він тимчасово прискорюється, що ускладнює гру та робить її більш динамічною.

- **Поступове ускладнення**  
  Після кожного успішного удару м’яч незначно збільшує швидкість, але в межах встановленого максимуму.

- **AI-суперник**  
  Права ракетка за замовчуванням керується комп’ютером. Штучний інтелект має обмежену швидкість реакції, що забезпечує збалансований рівень складності.

- **Ручне керування правою ракеткою**  
  За допомогою утримання клавіші **SHIFT** можна тимчасово перейти у режим гри *гравець проти гравця*.

- **Система станів гри**  
  Реалізовані стани меню, активної гри, паузи та завершення гри.
